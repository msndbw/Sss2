🗄️ قاعدة بيانات الطلبات

كل طلب يرسله الزبون من الموقع يُحفظ تلقائياً بريبو GitHub منفصل تماماً عن ريبو المنتجات (حماية إضافية). هذا يتيح للزبون والإدارة البحث عن أي طلب لاحقاً برقم الفاتورة.

  1. أنشئ ريبو GitHub جديد فارغ مخصص للطلبات فقط (مثال: khatwa-orders). يفضّل أن يكون عاماً (Public) ليعمل البحث من الموقع بدون توكن.
  2. تأكد أن خاصية Issues مفعّلة بإعدادات الريبو الجديد.
  3. أنشئ Personal Access Token (Fine-grained) بصلاحية Issues: Read & Write على هذا الريبو فقط — بدون أي صلاحية Contents.
  4. افتح ملف orders-config.js بمشروعك واملأ owner وrepo وwriteToken بالقيم، ثم انشره على GitHub Pages.

مسح إعدادات GitHub والـ PIN من هذا المتصفح فقط (لا يحذف أي شي من الريبو).
